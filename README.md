# Como Executar?

> Clone o Repositório:

Node version 20.10.0

Clone

```bash
git clone https://github.com/JonatanPaes/imhere.git
```

Para executar o projeto

```bash
cd imhere
```

Instale as dependências do projeto

```bash
npm install
```


# Mobile
Para executar o projeto

```sh
# Executar a aplicação
npm run start
```
